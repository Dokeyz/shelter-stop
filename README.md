# The Shelter Shop

## Purpose
The Shelter Shops purpose to increase views of adoptable pets, in order to maximize the rehoming rates of those friends in need.

## Built With
* HTML
* CSS - Bulma CSS Framework
* JavaScript
* JQuery
* Third Party APIs
